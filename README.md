# python_fastapi
Python FastAPI Project utilising JWT auth, get, post and delete Methods

Fast API based Backend Service to
-Register as a user
-Login as User
-Pulish and upload the Photo/Video
-Retrieve/Get owned Feed items
-Delete the published posts

1. Set up your custom Secrets in app/user.py
2. Need to fetch you own Public,Private Key for ImageKitio Account after successful registration on https://imagekit.io/dashboard/developer/api-keys
